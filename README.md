## Prerequisites

Make sure you have node.js and npm installed ( https://nodejs.org/en/ )

## Usage

1. Clone or download this repository to your local machine.
2. Traverse to it in a terminal.
3. Enter npm install
4. Enter npm start
5. Open browser and go to localhost:3000
